
# GoCRM
使用golang开发CRM  
前台DWZ 后端beego  

#安装
在gopath下
git clone https://github.com/xiongpf/GoCRM.git

#依赖包  
请执行以下命令:  

go get github.com/astaxie/beego  

go get github.com/go-sql-driver/mysql  

#数据库文件
在GoCRM/doc/go_crm.sql

#运行
```
cd GoCRM/  

go run main.go
```

##查看效果##
http://127.0.0.1:8025/crm/index
